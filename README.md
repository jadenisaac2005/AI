# Teach AI To Play Snake! Reinforcement Learning With PyTorch and Pygame

In this Python Reinforcement Learning I have followed teh Tutorial series from freeCodeCamp in which they teach an AI to play Snake! We build everything from scratch using Pygame and PyTorch.

You can find the tutorial I used here: [Playlist](https://www.youtube.com/playlist?list=PLqnslRFeH2UrDh7vUmJ60YrmWd64mTTKV)

- To install the requirements run the command ```pip install -r requirements.txt``` in your terminal in your root directory.
